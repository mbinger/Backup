1) Download and unpack the solution into some folder
2) Configure parameters in backup.bat:
	* Temp folder path
	* Destination folder path witch is being automatically copied to the cloud
	* Folders to be backuped
	* Password for archive protection
	* Etc
3) Download 7-zip console archiver from https://www.7-zip.org/ and place 7za.exe into same folder with backup.bat if you want to use archive password protection feature
4) Run schedule.bat to create a windows scheduler task for daily execution
